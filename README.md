# Getting Started
## Prerequisites
Please read "requirement.txt" or run:

```
conda env create -f environment.yml
```

## Usage
Please run:
```
python3 main_HomPINN.py
```

```
python3 main_HomPINN.py --data_dir ./data/obs_ex2 --num_sol 4 --num_obs 70 --num_col 100 --n_epoch 20000 --max_epoch 40000
```
